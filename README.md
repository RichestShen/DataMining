## 关于 `DataImporter` 的说明

为了方便进行数据处理, 在此处将常用库进行导入, 并进行了常用的设定, 解决了一些常用的问题, 例如中文显示, 中文负号等, 并给出了几组实用的配色方案.

使用时, 将此文件复制到目录下, 并在程序开头添加如下代码: `from DataImporter import *`.
